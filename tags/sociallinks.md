# Social links

Creates a ul/li list of links and icons to different social media websites

```html
<sf-sociallinks
    id="{{ component_id_1 }}"
    description="Social links"
    html-element="ul"
    class="sociallinks"
    caption="false"
>
    <li>
		<a href="%url%">
			<img src="/themes/clean/public/images/ui/%service%_icon.png" alt="%service%">
		</a>
	</li>
</sf-sociallinks>
```
